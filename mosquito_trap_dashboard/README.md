# Dự án: Dashboard theo dõi bẫy muỗi

## Mục tiêu
Xây dựng dashboard theo dõi:
- Tình trạng đặt bẫy muỗi
- Lượng muỗi xuất hiện tại từng bẫy
- So sánh theo khu vực đặt bẫy

## Tham chiếu giao diện

## Trạng thái
Draft - Initialized

## Gợi ý phạm vi dữ liệu ban đầu
- Danh sách bẫy: mã bẫy, vị trí, khu vực, ngày lắp
- Số liệu theo thời gian: ngày/giờ, số muỗi ghi nhận
- Tình trạng bẫy: hoạt động/lỗi/bảo trì
- Thuộc tính khu vực: quận/huyện, phường/xã, tọa độ

## Deliverables dự kiến
1. Bản mô tả yêu cầu dashboard
2. Data model (schema + nguồn dữ liệu)
3. Bộ KPI/chỉ số chính
4. Wireframe giao diện theo mẫu tham chiếu
5. Kế hoạch triển khai MVP
